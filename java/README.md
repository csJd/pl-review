# My Frequently Forgotten Facts (FFFs)
======

## 1. 访问权限

### 各修饰符访问权限如下表：
|Modifier|Class|Package|Out Subclass|Other|Total|
|:--:|:--:|:--:|:--:|:--:|:--:|
|`private`|Y|N|N|N|1|
|`default`|Y|Y|N|N|2|
|`protected`|Y|Y|Y|N|3|
|`public`|Y|Y|Y|Y|4|

[See doc here.](https://docs.oracle.com/javase/tutorial/java/javaOO/accesscontrol.html)
